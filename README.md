# Monitoramento de Rios (UFAM - IEAA)

Este projeto monitora os níveis dos rios Madeira (Humaitá) e Purus (Lábrea).

## Como rodar localmente
1. Instale o Node.js
2. No terminal, execute: `npm install`
3. Inicie o servidor: `npm start`
4. Acesse: `http://localhost:3000`

## Funcionalidades
- Dashboard público com gráfico de tendência (30 dias).
- Painel administrativo protegido por login.
- Exibição de estatísticas em cm.
